Installation.

Avant d'installer ODFAEG vous devez installer cwc (Les fichiers cmake sous obsolètes)
et vous devez aller sur la branche windows car la branche master n'est plus maintenue.
Une fois cwc d'installer il vous suffit de regarder le fichier build dans le dossier cwcODFAEGDEMO par exemple. Vous copier coller se fichier dans le dossier de votre projet et il vous suffit alors de remplacer ../Demo/ODFAEG-DEMO/ par le dossier de votre projet. Pour compiler et exécuter la démo il vous suffit de cliquer sur le fichier Make_For_Windows[Release]
